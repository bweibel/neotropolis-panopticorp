# Panopticorp Game Design Document

## Overview
Multi-layer hacking game where players infiltrate Panopticorp's systems through:
- Minigame challenges (hacking puzzles)
- Narrative progression (emails, chat messages)
- Easter egg discovery on corporate website

## Minigame Types
1. **Pattern Match** - Memory/sequence puzzles
2. **Minesweeper Variant** - Grid-based logic puzzle
3. **Network Pathfind** - Routing/connection puzzles
